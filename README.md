# Customer Data Cleaning and Marketing Segmentation

## Objective
Clean and prepare customer data to enable accurate segmentation and support data-driven marketing initiatives.

## Context
An e-commerce company is preparing the launch of a customer loyalty program and needs reliable, clean and structured customer data to design personalized marketing campaigns.

## Data
The dataset includes customer profiles with:
- Customer ID
- Name and age
- Favorite product categories
- Spending by category

The data contained inconsistencies, formatting issues and required validation.

## Analysis
The analysis focused on:
- Cleaning and standardizing customer names and ages
- Validating data consistency
- Calculating total spending per customer
- Preparing the dataset for future segmentation and KPI analysis

## Tools
- Python  
- Pandas  
- NumPy  
- Jupyter Notebook  

## Key Insights
- Data standardization is critical for reliable customer segmentation
- Spending patterns by category provide valuable inputs for marketing strategies
- The cleaned dataset is ready for advanced analytical use
